# 나만의 채용서비스 백엔드 서버

# 실행 방법


- 필요한 패키지 설치
```sh
yarn
```

- 서버 실행(배포용)
```sh
yarn start
```

- 서버 실행(개발용)
```sh
yarn dev
```
